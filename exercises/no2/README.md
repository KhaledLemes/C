# C Basics – Module 1 (Conditionals)

This folder contains small C programs created while learning **conditional logic**.

The exercises focus on using `if`, `else if`, `else`, and `switch` to control program flow based on different conditions.

## Topics Covered

- Conditional statements (`if`, `else if`, `else`)
- Logical operators (`&&`, `||`)
- Comparison operators
- `switch` statements
- Basic input validation
- Simple numeric checks (divisibility, ranges, etc.)
- Basic decision making in programs

## Purpose

These programs are simple practice exercises meant to reinforce how **decision-making works in C programs**.

They help build familiarity with:

- Evaluating expressions
- Handling multiple conditions
- Structuring control flow
