# C Basics – Module 1

This folder contains simple C programs written during **Module 1 lessons**.  
The goal is to practice fundamental programming concepts using the C language.

## Topics Covered

- Variable declaration and assignment
- Basic data types (`int`, `float`, `double`, `char`)
- Input and output using `printf` and `scanf`
- Arithmetic operations
- Updating and swapping variable values
- Simple calculations
- Conditional logic basics
- Loops (`for`)
- Basic problem solving

## Purpose

These programs are **small exercises meant for learning and experimentation**.  
They focus on understanding how basic C syntax and logic work.
